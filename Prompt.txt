# Antigravity Project Prompt

## Project Title

**CollabBridge AI: A Multi-Agent Platform for Matching Researchers with High-Impact Societal Projects**

---

# Project Vision

Build a production-quality AI multi-agent web application using modern agent engineering principles. The platform is designed for universities, NGOs, government agencies, research centers, and social enterprises.

The objective is to accelerate reconstruction and sustainable development by intelligently matching researchers with impactful projects.

The first deployment target is **Syria**, where the platform helps solve urgent post-war challenges by connecting project owners with researchers from Syria and around the world.

The application should demonstrate advanced AI agent design, multi-agent collaboration, structured reasoning, memory, explainability, and intelligent recommendations.

The entire user experience should be conversational instead of traditional forms.

3. Structured profile generation from conversations.
4. Explainable hybrid matching engine.
5. Team recommendation engine.
6. Proposal generation system.
7. Impact assessment module.
8. Analytics dashboard.
9. Synthetic datasets (50 researchers and 30 Syria-focused projects).
10. Clean, modular, production-ready code with documentation, tests, and Docker support.

The project should showcase modern AI agent engineering, emphasizing transparency, collaboration, and measurable societal impact. The design should make it easy to extend the platform to other countries facing reconstruction or sustainable development challenges after the initial Syria-focused deployment.
